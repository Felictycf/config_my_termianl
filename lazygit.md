### 🚀 LazyGit 核心快捷键指南

LazyGit 的界面分为左侧的四个面板（状态、文件、分支、提交）和右侧的主视图。

#### 1. 🧭 导航与通用
| 快捷键 | 作用 | 说明 |
| :--- | :--- | :--- |
| **`[`** / **`]`** | **切换面板** | 或者使用 `←` / `→` / `h` / `l` (Vim 风格) |
| **`↑`** / **`↓`** | **上下移动** | 或者使用 `k` / `j` (Vim 风格) |
| **`?`** | **打开帮助菜单** | **最重要的键！** 不记得快捷键时按它 |
| **`q`** | **退出** | 退出 LazyGit 回到终端 |
| **`@`** | **切换主视图** | 切换右侧窗口显示 diff 还是命令日志 |

#### 2. 📝 文件操作 (Files 面板 - 你截图中选中的区域)
这是你最常用的区域，用于处理变更。

| 快捷键 | 作用 | 说明 |
| :--- | :--- | :--- |
| **`Space` (空格)** | **暂存/取消暂存** (git add) | 选中文件按空格，变绿就是暂存了 |
| **`a`** | **暂存所有文件** | 相当于 `git add .` |
| **`d`** | **丢弃更改** (git checkout) | **慎用！** 会撤销你对文件的修改 |
| **`c`** | **提交** (git commit) | 按下后弹出输入框写 Commit Message |
| **`e`** | **编辑文件** | 用你配置的编辑器（如 vim）打开当前文件 |
| **`i`** | **添加到 .gitignore** | 快速忽略文件 |

#### 3. 🌿 分支与远程 (Local Branches 面板)
| 快捷键 | 作用 | 说明 |
| :--- | :--- | :--- |
| **`Space` (空格)** | **检出分支** (checkout) | 切换到当前选中的分支 |
| **`n`** | **新建分支** | 从当前位置创建新分支 |
| **`P`** (大写) | **推送** (git push) | 推送到远程仓库 |
| **`p`** (小写) | **拉取** (git pull) | 从远程仓库拉取 |

#### 4. 📜 历史提交 (Commits 面板)
| 快捷键 | 作用 | 说明 |
| :--- | :--- | :--- |
| **`Enter`** | **查看提交详情** | 可以在右侧看到该次提交改了什么 |
| **`s`** | **压缩提交** (Squash) | 选中两个提交进行合并 (交互式变基) |
| **`r`** | **重命名提交** | 修改 Commit Message |
